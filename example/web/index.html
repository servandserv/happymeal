Examples

Data adapters

Generate data adapters from XML Schema files with `happymeal` script.

Use data adapters to transform data in different formats (xml, json, markup array) to php data objects and vice versa.

  • Read XML from file, parse, validate without error and return as XML
  • Read XML from file, parse, change property to wrong value, validate with error and return as XML
  • Read XML from file, parse, validate without error and return as JSON
  • Read from markup array, validate without error and return as XML
